部署到 GitHub Pages 的办法 🚀

这个网页是纯静态项目：只有 HTML/CSS/JS，不需要 npm、不需要后端。只要开启 Pages，就可以访问。

已上传的文件

  • index.html

  • daily-plan.html
  • ge.html
  • ap-lit.html
  • ap-chem.html
  • calc.html
  • whap.html
  • styles.css / app.js / data.js

开启 Pages

  1. 进入 GitHub repo 的 Settings。
  2. 点击 Pages。
  3. Source 选择 Deploy from a branch。
  4. Branch 选择 main，folder 选择 /root。
  5. 保存后等待 1-3 分钟。

注意保存状态

Checklist 的保存依赖浏览器 localStorage。部署到 GitHub Pages 后，只要你一直用同一个浏览器打开同一个网址，勾选会保存。换设备前请先到互动清单页面导出 JSON。
