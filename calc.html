Calc BC：提前学习 limits，然后完成题包 📈

AP Calculus BC 的暑假作业是一份关于 limits 极限 的预习题包。你要看四个视频自学基本概念，然后完成 PDF 里的题，最后用红笔订正。

作业介绍：这份 Calc BC 暑假作业到底要你干什么？

这份作业的目标是让你开学前先接触 Calculus 的第一个核心概念：limit。Limit 不是普通代数题，它关心的是“当 x 靠近某个数时，函数值靠近什么”。所以你会练习从文字、图像、表格中判断极限。

学习部分

你需要打开老师给的 flippedmath 链接，按 Section 1 到 Section 4 看视频。视频就是这份作业的自学教材。

作业部分

视频看完后完成 PDF 里的题目。做完后用每个视频下面的 answer key 对答案，再用红笔订正错题。

你最终要交什么

  • 完成后的 Calc BC 暑假作业纸质版。
  • 题目要写完整。
  • 错题要用红笔订正。
  • 开学第一节 Calculus BC 课交给老师。

四个部分是什么

  1. Section 1: Can Change Occur at an Instant，瞬时变化的概念。
  2. Section 2: Defining Limits and Using Limit Notation，认识极限符号。
  3. Section 3: Limit Values from Graphs，从图像判断极限。
  4. Section 4: Limit Values from Tables，从表格判断极限。

做题小翻译

  • limit: x 靠近某个值时，f(x) 靠近什么。
  • f(2): x 真正等于 2 时函数值是多少。
  • 空心点: 那个点不算函数值，但可能表示 limit 靠近的位置。
  • 实心点: 真正的函数值。
  • does not exist: 左右靠近的值不同，或图像没有趋向同一个数。

推荐做法

先看视频，不要裸做。这份作业的目的就是让你用视频自学第一轮。
做题时用铅笔。先保证能改，订正时再用红笔。
图像题先看左右。从左边靠近和从右边靠近，如果同一个数，two-sided limit 存在。
表格题看趋势。不要只看 x 等于目标值那一格，要看左右两侧的 f(x) 趋向哪里。
最后红笔订正。老师明确要求对答案后用红笔改错。
