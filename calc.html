Calc BC：四个视频 + 四组 limits 题 📈

这门作业结构最清楚：看对应视频，做对应题，对答案，用红笔订正，开学第一节课交纸质版。它适合六月集中打穿。

你要完成什么

  1. 看 Section 1 视频：Can Change Occur at an Instant。
  2. 看 Section 2 视频：Defining Limits and Using Limit Notation。
  3. 看 Section 3 视频：Limit Values from Graphs。
  4. 看 Section 4 视频：Limit Values from Tables。
  5. 完成 PDF 里对应题目。
  6. 用网站视频下方 answer keys 对答案。
  7. 错题用红笔订正。
  8. 保留纸质版，第一节课交。

做题小翻译

  • limit: x 靠近某个值时，f(x) 靠近什么。
  • f(2): x 真正等于 2 时函数值是多少。
  • 空心点: 那个点不算函数值，但可能表示 limit 靠近的位置。
  • 实心点: 真正的函数值。
  • does not exist: 左右靠近的值不同，或图像没有趋向同一个数。

推荐做法

先看视频，不要裸做。这份作业的目的就是让你用视频自学第一轮。
做题时用铅笔。先保证能改，订正时再用红笔。
图像题先看左右。从左边靠近和从右边靠近，如果同一个数，two-sided limit 存在。
表格题看趋势。不要只看 x 等于目标值那一格，要看左右两侧的 f(x) 趋向哪里。
最后红笔订正。老师明确要求对答案后用红笔改错。
