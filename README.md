# Aoody Summer Homework Plan Web V3

一个离线可用的静态暑假作业计划网页。

## 功能

- 多页面作业说明
- 详细互动 checklist
- 自动保存勾选状态到 localStorage
- 支持导出/导入进度 JSON
- 6月10日到6月30日每日安排
- General English 选书指南
- WHAP 地图绘制攻略

## 本地打开

直接双击 `index.html`。

## GitHub Pages

把所有文件上传到 GitHub 仓库根目录，然后在 Settings -> Pages 中选择 Deploy from a branch, main, /root。
